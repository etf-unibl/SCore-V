Detailed Description

Data Memory (DMEM) entity for the SCore-V processor.

Initialised from a hex word file (one 32-bit word per line) at elaboration time. Memory size is derived automatically from the file (lines * 4 bytes). Supports byte-addressed asynchronous reads and synchronous writes of byte, halfword, or word width.
