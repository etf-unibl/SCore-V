Data Read and Sign Extension Logic.

Performs an asynchronous read from the byte-addressable DMEM. The process formats the output based on the requested data width and sign:

  • Byte (00): Loads 8 bits. Sign-extends if sign_i = '0', zero-extends if sign_i = '1'.
  • Halfword (01): Loads 16 bits. Sign-extends if sign_i = '0', zero-extends if sign_i = '1'.
  • Word (10): Loads 32 bits directly from four consecutive memory locations.
    Note
    This process is combinatorial and updates whenever address, control signals, or memory content changes.
    Warning
    Validates that the address is within [0, 252] to prevent out-of-bounds array access during simulation.
